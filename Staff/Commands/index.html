CD Hub Commands

Below is the list of available commands sorted by staff rank.

All Ranks

  • cd!warn user reason - Warn a user for a specific reason.
  • cd!mute user time(minutes) reason - Mute a user for a specific time with a reason.
  • cd!unmute user reason - Unmute a previously muted user with a reason.
  • cd!ki key - View information about a specific key.
  • cd!whois - View data for a CD Hub account.

Elite Moderator

  • cd!hwid user - Reset the device ID of a user's account.

Lead Developer & Owner

  • cd!genType Days Provider - Generate a key for CD Hub with specified type, days, and provider.
  • cd!ban user/key - Ban a user or account using either the user or key.
  • cd!bn user days reason - Ban a user from the server for a specific number of days with a reason.
